== Description ==
Plugin to generate front page (home page). Universal plugin for almost every theme. You can turn your blogging theme into one page layout with easy. It is lightweight but powerful tool to create awesome front page.

= Demo =
[Demo layout](http://front-page-builder.themes4wp.com) is included in the plugin. Activate the plugin and check your front page.

= How to setup plugin =
Go to Customizer - Front Page Builder Settings - Front Page Layout and drag & drop (or disable) sections. Go back and setup the section one by one

= Documentation =
[Documentation](http://demo.themes4wp.com/documentation/category/front-page-builder/) is available on Themes4WP documentation page.

== Installation ==
Simply install as a normal WordPress plugin and activate.

= After activating the plugin =
Go to Customizer - Front Page Builder Settings and setup your front page

= Deactivate plugin on front page =
Go to Customizer - Front Page Builder Settings - Front Page Layout and hit Force enable on front page button to disable the plugin on front page 

== Frequently Asked Questions ==
= Will the plugin work with my theme =
This builder is not theme based plugin. It will work with almost every theme. It is tested with 50+ major themes, however it is not possible to test the plugin with all existing themes and there could be some conflicts. If you find some issue, let us know via our contact form (describe your problem and tell us the theme name, that you are using with this plugin)

= Does this plugin generate more pages? =
No. This plugin generate only one page.

= Is possible to display the layout on custom page? =
Yes. Create a page and set template \"Front Page Builder Template\"
